#cts-link-analyzer

## Description

A commander CLI app

## Usage

To run, you need to first generate a link db

```node ./bin/cts-link-analyzer generate-link-db --server=<db_server> --port=<db_port> --user=<db_user> --passwd=<pass> <input_xlsx> <output_file>```
